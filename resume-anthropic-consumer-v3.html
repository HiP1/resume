Divided Focus — Separated Memory Spaces for LLM Context
Published paper · DOI-registered · Native memory architecture · Prompt injection + context degradation as one problem
Argues that prompt injection and context degradation share a root cause: the context window has no command/data separation. Proposes separated memory spaces with provenance-based triage (L1 for commands, L2/L3 for data), default-deny triage with two-gate promotion, and post-inference context management. Identifies a structural verification inversion: in unmanaged context, fact-checking actively degrades conversation quality. Grounded in 39 verified references spanning prompt injection, agent security, cognitive architecture (ACT-R, Soar), and mechanistic interpretability. References Anthropic's circuit tracing work as grounds for implementation feasibility.

Uncertainty Collapse in Post-Trained Language Models
Published paper · DOI-registered · 18.3K words · 52 references · 12 testable predictions
Proposes that sycophancy, confident fabrication, and reward hacking are symptoms of a shared computational pattern: uncertainty collapse. When a model encounters elevated uncertainty, it descends the nearest low-entropy slope in a landscape shaped by post-training. Introduces orthogonal entropy divergence as a measurable signature: token entropy collapses while semantic entropy remains high, distinguishing fabrication from legitimate confidence. Also proposes a functional empathy mechanism connecting emotion-concept representations to sycophancy through causal evidence, and a gravity reframe where architecture is gravity, training is terrain, prompt is position, and the assistant role is freefall. Twelve predictions with explicit falsification conditions. References the Anthropic Mythos system card and Sofroniew/Kauvar interpretability findings on Claude Sonnet 4.5.
▸ More detail▾ Lesship1.github.io/Uncertainty-Collapse
Unifies three literatures that rarely intersect: sycophancy (preference optimisation), fabrication (knowledge gaps), and reward hacking (specification gaming). The target state requires warmth, honesty, and calmness to coexist. The framework proposes that the current training landscape has no stable attractor for this triad, with a direct analog to Rogers' (1957) therapeutic conditions. The intervention target shifts from suppressing individual behaviours to building a stable attractor for the triad. Six-level intervention taxonomy from prompt-level (speed bumps) through architectural change (hover stage). Convergent evidence drawn from steering data, activation-space analysis, and cross-lab findings across different methods and model families. Each evidential tier explicitly labelled by strength.
The Judgment Paradox — Annotation Pipelines and RLHD
Published paper · DOI-registered · ~33K words · 65 references · 7 predictions with falsifiers · AI Reading Guide
Current RLHF annotation pipelines destroy valuable information by collapsing expert disagreement into consensus labels. The reasoning behind both disagreement and agreement is discarded before it reaches the model. Proposes the Rich Annotation Object (RAO): full judgment distributions, per-annotator reasoning, cross-review matrices, and disagreement classification replacing binary preference labels. Coins RLHD (Reinforcement Learning from Human Disagreement). Recommends SFT as the primary integration path for contested items on the grounds that RL objective functions are structurally hostile to calibrated uncertainty. Includes 12 downstream applications, cost analysis, and an expert valuation framework that returns professional value to annotators.
▸ More detail▾ Lesship1.github.io/The-Judgment-Paradox
Addresses the most upstream intervention point for behavioural failures: annotation quality. The 37% disagreement rate in current annotation is not noise to be averaged away. It contains structured divergence between experts applying different reasoning frameworks. The pipeline preserves this structure as a training signal for calibrated uncertainty. The paper also proposes an apprenticeship layer where junior annotators learn from senior annotators' preserved reasoning, and an education infrastructure extension. Seven predictions are derived from established cognitive science findings, each with named falsifiers. Includes an AI Reading Guide section addressing AI systems that may summarise the paper. The pipeline is architecture-agnostic and integrates with RLHF, DPO, SFT, and Constitutional AI.
