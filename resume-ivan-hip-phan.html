Strategic Forgetting — LLM Context Management Protocol
Proposes tiered context persistence for LLM inference: differentiating retention policies for retrieved content instead of treating all tokens as equally persistent. Defines the failure mode as "involuntary eidetic retention followed by catastrophic amnesia" and specifies a protocol with design invariants, state machines, and an identified security risk (semantic cache poisoning through summarisation).
